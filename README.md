# Task-05: Neural Style Transfer

## Overview
This project demonstrates Neural Style Transfer, a deep learning technique that combines the artistic style of one image with the content of another image. The model generates a new image that preserves the original content while applying the selected artistic style.

## Objective
- Apply the style of one image to another using a pre-trained neural network.
- Generate visually appealing stylized images while preserving content.

## Features
- Content and style image processing.
- Deep learning-based style transfer.
- High-quality stylized image generation.
- Easy-to-use implementation.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- OpenCV (optional)
